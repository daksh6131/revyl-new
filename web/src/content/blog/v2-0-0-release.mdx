# What's New in v2.0.0

This major release brings a completely redesigned test editor and real-time device streaming.

## New Features

- **New graph-based test editor with drag-and-drop** - Create tests visually with our new node-based editor
- **Real-time device streaming via WebRTC** - Watch your tests execute live
- **Natural language test instructions** - Write test instructions in plain English

## Bug Fixes

- Improved element detection accuracy
- Fixed iOS simulator connection issues

## Breaking Changes

- Updated API response format for test results
